Инструкция по установке PHP 8
Данная инструкция покажет вам, как установить PHP 8.
PHP — это интерпретируемый язык программирования, и для выполнения php-скриптов (то есть файлов с кодом на PHP) потребуется интерпретатор PHP
По состоянию на февраль 2024 года последний выпуск — PHP 8.3.
1. Установка и настройка PHP 8 в Ubuntu Linux
1.1. Проверка версии Ubuntu
Откройте приложение Терминал (Terminal).
Проверьте версию операционной системы следующей командой:
lsb_release -a
Наша цель — установить PHP версии 8.1 или выше, при этом:
в репозиториях Ubuntu версии 20.04 LTS поставляется PHP 7.2
в репозиториях Ubuntu версий 22.04 LTS, 23.04 поставляется PHP 8.1
в репозитории Ubuntu версии 23.10 поставляется PHP 8.2
Если в вашей версии Ubuntu нет достаточно высокой версии PHP, можно подключить сторонний репозиторий ppa:ondrej/php, как показано ниже.
1.2. Подключение репозитория ppa:ondrej/php
Репозиторий ppa:ondrej/php долгое время сопровождает чешский программист Ondřej Surý.
Если вас устраивает версия PHP из репозиториев Ubuntu, пропустите данный раздел.
По состоянию на 2023-02-10 сторонний репозиторий ppa:ondrej/php содержит PHP 8.3.
Выполните в Терминале:
# Установка репозитория
# Потребуется подтвердить, нажав Enter
sudo add-apt-repository ppa:ondrej/php
# Обновление данных о пакетах
sudo apt-get update
1.3 Установка PHP 8.2
Выполните в терминале:
# Обновление данных о пакетах
sudo apt-get update
# Установка пакетов
sudo apt-get install php-cli php-mysql php-mbstring
# Проверка версии PHP
php --version
Установленная версия PHP должна быть 8.3, 8.2 или 8.1.
2. Установка и настройка PHP 8 на Windows
2.1 Загрузка PHP для Windows
Откройте страницу https://windows.php.net/download и загрузите Zip-архив с последней версией PHP:

2.2. Распаковка и настройка PHP
Распакуйте Zip-архив с предыдущего шага в удобное место на диске, например, в C:\php\php-8.2-nts\
Далее предполагается, что каталог с распакованным дистрибутивом PHP называется C:\php\php-8.2-nts\
После этого:
Откройте каталог C:\php\php-8.2-nts\ и скопируйте файл php.ini-development в файл php.ini
Добавьте путь к каталогу C:\php\php-8.2-nts\ в системную переменную PATH
Откройте командную строку и выполните: php --version
Ожидаемый результат: интерпретатор PHP вывел свою версию
В командной строке выполните: php -m
Ожидаемый результат: интерпретатор PHP вывел список расширений PHP (PHP extensions)
Если список модулей пуст, значит, вы не настроили файл php.ini
Решение проблемы с VCRUNTIME*.dll
Если при попытке запуска php появляется системное окно ошибки, которое сообщает об отсутствии VCRUNTIME*.dll, то вам нужно загрузить и выполнить установку Visual C++ Redistributable:
https://aka.ms/vs/16/release/VC_redist.x64.exe
3.3. Включение модуля pdo_mysql
| Что такое PDO: PHP: PDO - Manual
В рамках курса вам потребуется работать с MySQL из PHP с помощью PHP-PDO.
Для этого в выводе команды php -m должны быть следующие модули:
PDO
pdo_mysql
Второй модуль по умолчанию отключён, его следует включить вручную:
Откройте файл php.ini в Visual Studio Code
Найдите строку ;extension_dir = "ext" и уберите комментирующий символ ';' (точка с запятой)
Найдите строку ;extension=pdo_mysql и уберите комментирующий символ
Сохраните файл php.ini и в терминале выполните: php -m
Ожидаемый результат: команда php -m вывела в списке модули PDO и pdo_mysql.
3. Запуск встроенного веб-сервера
Выполните заранее инструкцию по установке VSCode
3.1. Подготовка проекта
В Visual Studio Code с помощью меню “Open Folder…” откройте любой проект с PHP скриптами
Если у вас ещё нет проекта, то в диалоге выбора каталога создайте новый каталог и выберите его
Если в проекте нет файла index.php, добавьте его, используя один из шаблонов ниже
Минимальный пример PHP скрипта:
<?php
echo 'Hello, PHP!';
Минимальный пример PHP скрипта, сделанного в стиле HTML-шаблона со вставками кода:
<html>
<head>
<title>PHP Test Page</title>
</head>
<body>
<h1><?= 'Hello, PHP!' ?></h1>
<body>
</html>
3.2. Запуск встроенного веб-сервера
| Что такое php builtin webserver: PHP: Встроенный веб-сервер - Manual
В современных версиях PHP интерпретатор содержит встроенный однопоточный веб-сервер, пригодный для учебных целей.
Как это использовать:
Откройте терминал в каталоге проекта и выполните: php -S 127.0.0.1:8000
Откройте в браузере: http://127.0.0.1:8000/
Если в каталоге проекта есть скрипт index.php, он будет выполнен и вы увидите в браузере его вывод.
Если у вас нет скрипта index.php, вы увидите следующее:

Можно выполнить любой другой PHP скрипт в каталоге проекта, указав имя скрипта (а точнее его относительный путь) в качестве URL Path. 
Пример URL: http://127.0.0.1:8000/hello.php
Этот URL соответствует скрипту hello.php в каталоге проекта
